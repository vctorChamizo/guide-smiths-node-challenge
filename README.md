# Guide Smiths | Node Challenge

## Setup ⚙️

---

### Run application 🖥

```
git clone git@github.com:vctorChamizo/guide-smiths-node-challenge.git

yarn install && yarn run dev
```

### Run tests 🧪

```
yarn run test
```

### Run with Docker 🐳

```
docker-compose up
```
